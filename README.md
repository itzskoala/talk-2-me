# talk-2-me 🤖

An AI-powered conversational agent built with OpenAI and Gradio that simulates a personalized chatbot (“AI version of me”) with memory, evaluation, and tool use.

---

## 🚀 TRY IT HERE!

> (https://huggingface.co/spaces/holasoykoalo7/ai-me)

---

## ✨ Features

- Chat interface built with Gradio
- Powered by OpenAI GPT models
- Personalized system prompt (acts as “me”)
- Loads personal context from PDFs + summary files
- Tool calling (user tracking + unknown question logging)
- Pushover notifications for leads/interactions
- Response evaluator (quality control layer)
- Auto retry with feedback-based regeneration

---

## 🧱 Tech Stack

- Python 3.10+
- Gradio
- OpenAI API
- PyPDF
- Pydantic
- Requests
- Python-dotenv
